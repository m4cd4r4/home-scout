### Added

- Initial repository scaffold
- ROS 2 workspace with 7 packages (interfaces, bringup, voice, vision, nav, memory, faces)
- STM32H5 motor control firmware (PlatformIO)
- ESP32-CAM streaming firmware (PlatformIO)
- Phase 1-6 build guide stubs
- Privacy, safety, and security documentation
- GitHub Actions CI (ROS 2, firmware, lint, docs, STL validation)
- Docker development environment
- Bill of Materials with three build tiers
- Personality configuration system (YAML)
- Simulation environment (Gazebo) stub
- Object memory SQLite schema
- CAN-FD communication protocol between Qualcomm and STM32H5
