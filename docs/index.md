# Home Scout Documentation

Welcome to the Home Scout documentation. Scout is a privacy-first home companion robot built on the [Arduino VENTUNO Q](https://www.arduino.cc/product-ventuno-q/).

## Build Phases

| Phase | Name | What You Get |
|-------|------|-------------|
| 1 | [Scout Can Talk](build-guides/phase-1-voice.md) | Voice assistant with personality |
| 2 | [Scout Can See](build-guides/phase-2-vision.md) | Object detection via VLM |
| 3 | [Scout Can Move](build-guides/phase-3-mobility.md) | Autonomous wheeled rover |
| 4 | [Scout Remembers](build-guides/phase-4-memory.md) | "Where are my keys?" |
| 5 | [Scout Knows Us](build-guides/phase-5-faces.md) | Face recognition + greetings |
| 6 | [Sky Eye](build-guides/phase-6-drone.md) | Optional drone module |

## Privacy

All processing happens on-device. Scout never connects to the internet. See [PRIVACY.md](https://github.com/m4cd4r4/home-scout/blob/main/PRIVACY.md).
