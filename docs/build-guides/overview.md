# Build Guide Overview

## Before You Start

1. **Read [SAFETY.md](https://github.com/YOUR_USERNAME/home-scout/blob/main/SAFETY.md)** before building anything
2. **Buy parts for one phase at a time** - see [BOM.md](https://github.com/YOUR_USERNAME/home-scout/blob/main/BOM.md)
3. **Each phase is standalone** - you can stop after any phase and have a working robot

## Tools Required

- Soldering iron (for Phase 3+)
- Multimeter
- Wire strippers
- Hex driver set (M2.5, M3)
- 3D printer access (optional - off-the-shelf chassis available)
- USB-C cable
- MicroSD card (for initial Ubuntu flash)

## Phase Order

Build in order. Each phase builds on the previous one.

| Phase | Time | Cost | Difficulty |
